# 时间追踪器

一个基于网页的时间追踪应用，帮助用户记录和分析他们的时间使用情况。

## 功能特点

- 实时时间追踪
- 活动记录和管理
- 数据可视化统计
- 年度统计表
- 数据导入导出
- 响应式设计

## 技术栈

- HTML5
- CSS3
- JavaScript
- Chart.js（用于数据可视化）

## 快速开始

1. 克隆仓库：
```bash
git clone https://github.com/kexin94yyds/Attention-artifact.git
```

2. 进入项目目录：
```bash
cd Attention-artifact
```

3. 启动本地服务器：
```bash
python3 -m http.server 8000
```

4. 在浏览器中访问：
```
http://localhost:8000
```

## 使用说明

1. 输入活动名称并点击"开始记录"来开始追踪时间
2. 使用统计视图查看不同类型的数据分析
3. 查看年度统计表了解长期时间使用情况
4. 使用导入/导出功能备份或迁移数据

## 项目结构

- `index.html` - 主页面
- `styles.css` - 主样式表
- `combined.js` - 主要JavaScript逻辑
- `chart_functions.js` - 图表相关功能
- `stats_functions.js` - 统计功能
- `annual_table.js` - 年度表格功能
- `export_import.js` - 数据导入导出功能
